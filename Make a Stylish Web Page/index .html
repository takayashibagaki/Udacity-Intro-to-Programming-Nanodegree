DOM

DOM - Document Object Model - a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.

CSS

CSS is code written to control the "style" of HTML elements. It is common to describe it in a css file separately from the HTML file and create it as a link in the HTML file. CSS can enrich HTML such as designation of character size, color, size and display specification of image
